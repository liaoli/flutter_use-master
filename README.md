# 打包操作

- android（关闭R8压缩）
  - flutter build apk --no-shrink
- ios（生成 .app ，然后 在 XCode 执行 打包 操作）
  - flutter build ios --release
- web：使用CanvasKit模式
  - flutter build web --web-renderer canvaskit
